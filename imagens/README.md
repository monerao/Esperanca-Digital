# 🌟 Esperança Digital – Plataforma Inteligente de Apoio Social

![GitHub repo size](https://img.shields.io/github/repo-size/seu-usuario/Esperanca-Digital)
![GitHub license](https://img.shields.io/github/license/seu-usuario/Esperanca-Digital)

**Esperança Digital** é uma plataforma web que utiliza Inteligência Artificial para conectar pessoas vulneráveis a oportunidades e serviços sociais. Desenvolvida como projeto integrador, demonstra o uso de tecnologias modernas para causar impacto social.

## 📋 Funcionalidades

- 🏠 **Página inicial** com design moderno e chamada para ação.
- 👨‍💻 **Sobre o projeto** – missão, visão e valores.
- 🤖 **Seção de IA** – explicação sobre como a inteligência artificial é aplicada.
- 📞 **Contato** – formulário com validação em JavaScript.
- 👥 **Equipe** – apresentação dos membros.
- ❓ **FAQ** – perguntas frequentes.
- 🔐 **Login e Cadastro** – telas funcionais para demonstração.
- 📱 **Responsivo** – adaptação para celular e tablet.
- ✨ **Animações** – transições suaves e efeitos visuais.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** – estrutura semântica.
- **CSS3** – estilização, layout grid/flexbox e animações.
- **JavaScript (ES6)** – interatividade, validação e menu mobile.
- **Font Awesome** – ícones vetoriais.
- **Git/GitHub** – controle de versão.
